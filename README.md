### NetFlow Analyzer

End-to-end network traffic analysis pipeline for anomaly detection and traffic classification.

Built with **Python, Docker, NFStream, Scapy, Scikit-learn, Streamlit and Plotly**. The project captures PCAP files in an isolated Docker lab, reconstructs bidirectional flows, extracts Layer 3 / Layer 4 features and applies machine learning models to classify normal traffic, SYN scan, UDP scan, ICMP flood and port sweep activity.

Achieved **96.97% test accuracy** and **96.89% weighted F1-score**.

**Repository:** [NetFlow-Analyzer](https://github.com/angelperezcastro/NetFlow-Analyzer)

### Vector Search Benchmark with PostgreSQL & pgvector

Benchmarking project for semantic vector search using PostgreSQL and pgvector.

Built with **PostgreSQL, pgvector, Docker, Python, Sentence Transformers, HNSW and IVFFlat**. The project compares exact K-NN search, HNSW and IVFFlat indexes across latency, recall and stability using 384-dimensional text embeddings.

**Repository:** [pgvector-search-benchmark](https://github.com/angelperezcastro/pgvector-search-benchmark)

### Credit Card Fraud Detection with PySpark

Machine learning workflow for fraud detection on highly imbalanced credit card transaction data.

Built with **Python, PySpark, Spark MLlib and Jupyter Notebook**. The project covers data cleaning, exploratory analysis, feature assembly, scaling, model training and evaluation using Logistic Regression and Random Forest models.

### AI-Generated Image Detection

Computer vision project for classifying real vs AI-generated images.

Built with **Python, PyTorch, ConvNeXt-Tiny and Grad-CAM**. The project includes model fine-tuning, validation, single-image inference and visual interpretability through Grad-CAM heatmaps.

Achieved **85.93% accuracy** and **85.92% macro F1-score** on a 600-image validation set.

**Repository:** [AI-Image-Detection](https://github.com/angelperezcastro/AI-Image-Detection)
